# Guía de Desarrollo - Credibanco App Angular

## 🛠️ Configuración del Entorno

### Prerrequisitos
- Node.js 18+
- npm o yarn
- Git

### Instalación
```bash
git clone https://github.com/giovanemere/credibanco-app-angular.git
cd credibanco-app-angular
npm install
```

## 🧪 Testing

```bash
# Ejecutar tests
npm test
```
